How to Connect OpenClaw to Email: Complete Setup Guide

Connect OpenClaw to Email and get a powerful AI assistant right where you already communicate. This guide walks you through every step ‚Äî from initial setup to sending your first message.

What You'll Need

  • An email account with IMAP/SMTP access
  • App password or application-specific credentials
  • OpenClaw installed on a VPS or local machine

Step 1: Set Up an Email Account for Your Bot

Create a dedicated email address for OpenClaw, or use an existing one. Popular options:

  • Gmail ‚Äî enable "App Passwords" (requires 2FA enabled). Go to Google Account ‚Üí Security ‚Üí App Passwords and generate one.
  • Custom domain ‚Äî set up a mailbox like claw@yourdomain.com via your hosting provider
  • Fastmail, ProtonMail Bridge, or any IMAP provider

For Gmail, the IMAP server is imap.gmail.com and SMTP is smtp.gmail.com.

Step 2: Configure OpenClaw for IMAP/SMTP

Add the email channel to your config:

{
  "channels": {
    "email": {
      "imap": {
        "host": "imap.gmail.com",
        "port": 993,
        "user": "your-bot@gmail.com",
        "password": "YOUR_APP_PASSWORD",
        "tls": true
      },
      "smtp": {
        "host": "smtp.gmail.com",
        "port": 465,
        "user": "your-bot@gmail.com",
        "password": "YOUR_APP_PASSWORD",
        "secure": true
      },
      "pollIntervalMs": 30000,
      "allowedSenders": ["you@example.com"]
    }
  }
}

The pollIntervalMs setting controls how often OpenClaw checks for new emails (30 seconds above).

Step 3: Set Up Email Filtering (Optional)

You probably don't want OpenClaw responding to every email. Configure filters:

  • allowedSenders ‚Äî only process emails from specific addresses
  • Subject prefix ‚Äî only process emails with a specific subject prefix like [claw]
  • Label/folder filtering ‚Äî in Gmail, use a filter to label certain emails, then configure OpenClaw to only read that label

Step 4: Start and Test

Start OpenClaw:

openclaw gateway start

Send an email to your bot's address from an allowed sender. OpenClaw will read it, process it, and reply via email. Check your inbox for the response.

What Can You Do with OpenClaw + Email?

Popular Use Cases

  • AI email assistant that drafts replies to common inquiries
  • Automated customer support via email
  • Email triage ‚Äî summarize, categorize, and prioritize your inbox
  • Scheduled email reports and digests

Platform Details

Email is universal ‚Äî everyone has it, every business uses it. Key advantages of the OpenClaw email integration:

  • Universal reach ‚Äî works with anyone who has an email address
  • No API approval needed ‚Äî IMAP/SMTP is an open standard
  • Asynchronous ‚Äî perfect for tasks that don't need instant responses
  • Attachment support ‚Äî process PDFs, images, spreadsheets attached to emails
  • Works with any provider ‚Äî Gmail, Outlook, Fastmail, self-hosted, anything with IMAP
