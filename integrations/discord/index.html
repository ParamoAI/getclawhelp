How to Connect OpenClaw to Discord: Complete Setup Guide

Connect OpenClaw to Discord and get a powerful AI assistant right where you already communicate. This guide walks you through every step ‚Äî from initial setup to sending your first message.

What You'll Need

  • A Discord account
  • A server where you have admin/manage permissions
  • OpenClaw installed on a VPS or local machine
  • Bot token from the Discord Developer Portal

Step 1: Create a Discord Application and Bot

Go to the Discord Developer Portal and click New Application. Name it (e.g., "OpenClaw") and go to the Bot tab.

Click Add Bot, then:

  • Copy the bot token ‚Äî you'll need this for OpenClaw
  • Enable Message Content Intent under Privileged Gateway Intents (required to read message content)
  • Enable Server Members Intent if you want user info

Step 2: Invite the Bot to Your Server

Go to OAuth2 ‚Üí URL Generator. Select these scopes:

  • bot
  • applications.commands

Under Bot Permissions, select:

  • Send Messages, Read Message History, Add Reactions, Embed Links, Attach Files, Use Slash Commands

Copy the generated URL and open it in your browser. Select your server and authorize the bot.

Step 3: Configure OpenClaw

Add Discord to your OpenClaw config:

{
  "channels": {
    "discord": {
      "botToken": "YOUR_DISCORD_BOT_TOKEN",
      "allowedGuilds": ["YOUR_SERVER_ID"],
      "allowedChannels": ["CHANNEL_ID_1", "CHANNEL_ID_2"]
    }
  }
}

To get server and channel IDs: enable Developer Mode in Discord settings (User Settings ‚Üí Advanced ‚Üí Developer Mode), then right-click on servers/channels to copy their IDs.

Step 4: Start and Test

Start OpenClaw:

openclaw gateway start

Go to a channel where your bot has access and send a message mentioning it (e.g., @OpenClaw what's the weather?). The bot should respond in the channel.

You can also DM the bot directly for private conversations.

What Can You Do with OpenClaw + Discord?

Popular Use Cases

  • Community management bot for Discord servers
  • AI-powered moderation assistant
  • Gaming community helper ‚Äî look up stats, schedules, and info
  • Team collaboration bot for private Discord servers

Platform Details

Discord is hugely popular for communities and increasingly for business teams. Key features for OpenClaw:

  • No public URL needed ‚Äî Discord uses a WebSocket gateway, so your bot connects outbound
  • Thread support ‚Äî keep conversations organized
  • Slash commands ‚Äî register custom /commands for structured interactions
  • Rich embeds ‚Äî beautiful formatted messages with colors, images, and fields
  • Free ‚Äî no per-message costs from Discord

Note: Discord requires the Message Content Intent for bots in 100+ servers. For private/small servers, it's automatically available.
