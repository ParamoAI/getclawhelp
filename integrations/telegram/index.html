How to Connect OpenClaw to Telegram: Complete Setup Guide

Connect OpenClaw to Telegram and get a powerful AI assistant right where you already communicate. This guide walks you through every step ‚Äî from initial setup to sending your first message.

What You'll Need

  • A Telegram account
  • A bot token from @BotFather
  • OpenClaw installed on a VPS or local machine
  • A public HTTPS endpoint (for webhooks only)

Step 1: Create a Telegram Bot via BotFather

Open Telegram and search for @BotFather. Send /newbot and follow the prompts to name your bot. BotFather will give you a bot token ‚Äî a long string like 7123456789:AAHx.... Save this token securely; you'll need it in the next step.

Optionally, send /setdescription and /setabouttext to customize how your bot appears to users.

Step 2: Configure OpenClaw's Telegram Channel

In your OpenClaw config file (openclaw.json), add the Telegram channel:

{
  "channels": {
    "telegram": {
      "botToken": "YOUR_BOT_TOKEN_HERE",
      "allowedUsers": ["your_telegram_user_id"],
      "polling": true
    }
  }
}

Set polling: true for simple setups (no public URL needed). For production, use webhooks with a public HTTPS endpoint instead.

Step 3: Set Up Webhooks (Production)

For a production setup, configure a webhook so Telegram pushes messages to your server instead of OpenClaw polling for them:

openclaw gateway config set telegram.webhook.url https://your-domain.com/telegram/webhook
openclaw gateway restart

You'll need a valid SSL certificate. If you're running on a VPS with Nginx, use Let's Encrypt for free certificates.

Step 4: Start OpenClaw and Test

Start (or restart) OpenClaw:

openclaw gateway start

Open your bot in Telegram and send a message. Your OpenClaw instance should respond. If it doesn't, check openclaw gateway status for errors.

What Can You Do with OpenClaw + Telegram?

Popular Use Cases

  • Personal AI assistant accessible from your phone anywhere
  • Group chat bot that answers team questions and runs automations
  • Customer support bot for Telegram-based communities
  • Automated notifications ‚Äî server alerts, calendar reminders, sales updates

Platform Details

Telegram is one of the most popular channels for OpenClaw because of its excellent bot API. Key advantages:

  • No approval process ‚Äî create a bot instantly via BotFather
  • Rich media support ‚Äî send images, files, buttons, and inline keyboards
  • Group chat support ‚Äî add your bot to any group
  • Free and unlimited ‚Äî no per-message costs from Telegram

Telegram's Bot API supports both long polling and webhooks. Polling is easier to set up (no public URL required), while webhooks are more efficient for production use.
