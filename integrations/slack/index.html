How to Connect OpenClaw to Slack: Complete Setup Guide

Connect OpenClaw to Slack and get a powerful AI assistant right where you already communicate. This guide walks you through every step ‚Äî from initial setup to sending your first message.

What You'll Need

  • A Slack workspace (free or paid)
  • Admin permissions to install apps
  • OpenClaw installed on a VPS or local machine
  • Bot and app-level tokens from Slack

Step 1: Create a Slack App

Go to api.slack.com/apps and click Create New App. Choose "From scratch" and give it a name (e.g., "OpenClaw"). Select your workspace.

Under OAuth & Permissions, add these bot token scopes:

  • chat:write ‚Äî send messages
  • channels:history ‚Äî read channel messages
  • channels:read ‚Äî list channels
  • im:history ‚Äî read DMs
  • im:write ‚Äî send DMs
  • reactions:write ‚Äî add emoji reactions
  • users:read ‚Äî look up user info

Step 2: Install the App and Get Tokens

Click Install to Workspace and authorize. You'll get two important values:

  • Bot User OAuth Token (starts with xoxb-)
  • App-Level Token ‚Äî create one under Basic Information ‚Üí App-Level Tokens with the connections:write scope (starts with xapp-)

Enable Socket Mode under Settings ‚Äî this lets OpenClaw connect without a public URL.

Step 3: Configure OpenClaw

Add the Slack channel to your OpenClaw config:

{
  "channels": {
    "slack": {
      "botToken": "xoxb-YOUR-BOT-TOKEN",
      "appToken": "xapp-YOUR-APP-TOKEN",
      "signingSecret": "YOUR_SIGNING_SECRET"
    }
  }
}

The signing secret is found under Basic Information in your Slack app settings.

Step 4: Enable Event Subscriptions

Under Event Subscriptions, subscribe to these bot events:

  • message.channels ‚Äî messages in public channels
  • message.im ‚Äî direct messages
  • app_mention ‚Äî when someone @mentions your bot

With Socket Mode enabled, you don't need a request URL. Start OpenClaw and it connects automatically:

openclaw gateway start

What Can You Do with OpenClaw + Slack?

Popular Use Cases

  • Team AI assistant that answers questions in any channel
  • Automated standup reports and meeting summaries
  • Sales and CRM notifications piped into Slack channels
  • Internal knowledge base ‚Äî ask your bot instead of searching docs

Platform Details

Slack is the most feature-rich integration for OpenClaw in a business context. Key advantages:

  • Socket Mode ‚Äî no public URL needed, works behind firewalls
  • Thread support ‚Äî bot replies stay organized in threads
  • Rich formatting ‚Äî markdown, code blocks, buttons, modals
  • Workspace-wide access ‚Äî one bot serves your entire team

Slack's API has rate limits (roughly 1 message per second per channel). OpenClaw handles this automatically with built-in queuing.
